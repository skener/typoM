Override Meta Tags
^^^^^^^^^^^^^^^^^^^

As in TYPO3 v9 LTS recommended, we use the MetaTag Api to add our Meta Tags. You can override them easily. Take a look
at the `TYPO3 Core API <https://docs.typo3.org/typo3cms/CoreApiReference/ApiOverview/MetaTagApi/Index.html>`_.

Furthermore we provide hooks to override the hreflang and canocial tags.
